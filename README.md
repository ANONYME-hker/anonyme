# ✨ Site pour Oceanne - Début de Magie

Un site web interactif créé pour célébrer ce moment spécial entre vous et Oceanne.

## 🌟 Fonctionnalités

- **Animation de fond** : Gradient animé avec des étoiles scintillantes
- **Symboles flottants** : Étoiles et symboles magiques qui flottent continuellement 
- **Texte animé** : Effet machine à écrire pour "Ce baiser m'a tout changé 💕"
- **Messages spéciaux** : Messages qui reflètent cette étape de votre relation
- **Bouton interactif** : Différentes surprises à chaque clic
- **Galerie de qualités** : Met en valeur les qualités que vous admirez chez elle
- **Effets visuels** : Feux d'artifice, explosions d'étoiles, arc-en-ciel
- **Effet de souris** : Traînée d'étoiles qui suit le curseur
- **Design responsive** : Fonctionne sur tous les appareils

## 🎯 Comment utiliser

1. Ouvrir `index.html` dans un navigateur web
2. Cliquer sur le bouton "Découvre plus" pour découvrir les surprises
3. Cliquer sur les qualités dans la galerie pour voir des messages spéciaux
4. Cliquer sur la note de musique 🎵 pour activer/désactiver la musique
5. Déplacer la souris pour voir des traînées d'étoiles

## 💝 Messages adaptés à votre situation

Le site contient des messages parfaits pour cette étape :
- "Ce baiser m'a tout changé 💕"
- "Notre connexion est magique"
- "Ton sourire me fait fondre"
- "Ce moment était parfait"
- "J'ai envie de découvrir la suite"

## 🎨 Technologies utilisées

- **HTML5** : Structure sémantique
- **CSS3** : Animations avancées, gradients, effets visuels
- **JavaScript** : Interactions, animations dynamiques, effets spéciaux
- **Google Fonts** : Polices élégantes (Dancing Script, Poppins)

## 🚀 Surprises cachées

- **1er clic** : Explosion d'étoiles et message sur ce moment magique
- **2ème clic** : Étoiles arc-en-ciel
- **3ème clic** : Feux d'artifice pour célébrer
- **4ème clic** : Message spécial sur le baiser
- **5ème clic+** : Explosion massive d'étoiles

## 📱 Compatible avec

- Desktop (Chrome, Firefox, Safari, Edge)
- Mobile (iOS Safari, Chrome Mobile)
- Tablettes
