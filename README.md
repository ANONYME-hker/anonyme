# 💕 Site Romantique pour Oceanne

Un site web interactif et romantique créé avec amour pour Oceanne.

## 🌟 Fonctionnalités

- **Animation de fond** : Gradient animé avec des étoiles scintillantes
- **Cœurs flottants** : Des cœurs qui flottent continuellement 
- **Texte animé** : Effet machine à écrire pour le message principal
- **Messages d'amour** : Plusieurs messages qui apparaissent avec des animations
- **Bouton interactif** : Différentes surprises à chaque clic
- **Galerie de souvenirs** : Photos interactives avec des messages personnalisés
- **Effets visuels** : Feux d'artifice, explosions de cœurs, arc-en-ciel
- **Effet de souris** : Traînée d'étoiles qui suit le curseur
- **Design responsive** : Fonctionne sur tous les appareils

## 🎯 Comment utiliser

1. Ouvrir `index.html` dans un navigateur web
2. Cliquer sur le bouton "Clique ici" pour découvrir les surprises
3. Cliquer sur les photos de la galerie pour voir des messages spéciaux
4. Cliquer sur la note de musique 🎵 pour activer/désactiver la musique
5. Déplacer la souris pour voir des traînées d'étoiles

## 💝 Messages d'amour

Le site contient plusieurs messages romantiques qui apparaissent progressivement :
- "Je t'aime de tout mon cœur 💕"
- "Tu es la lumière de ma vie"
- "Ton sourire illumine mes jours"
- "Chaque moment avec toi est un trésor"
- "Tu es mon rêve devenu réalité"

## 🎨 Technologies utilisées

- **HTML5** : Structure sémantique
- **CSS3** : Animations avancées, gradients, effets visuels
- **JavaScript** : Interactions, animations dynamiques, effets spéciaux
- **Google Fonts** : Polices romantiques (Dancing Script, Poppins)

## 🚀 Surprises cachées

- **1er clic** : Explosion de cœurs et message surprise
- **2ème clic** : Cœurs arc-en-ciel
- **3ème clic** : Feux d'artifice romantiques
- **4ème clic** : Message spécial et changement du bouton
- **5ème clic+** : Explosion massive de cœurs

## 📱 Compatible avec

- Desktop (Chrome, Firefox, Safari, Edge)
- Mobile (iOS Safari, Chrome Mobile)
- Tablettes
